分词
========
* 下载模型
在nlp-tf/nlp下
```python 
#coding=utf-8
import nlp.download as dl
model="seg"
dl.get_model(data)

```
* 分词测试
在nlp-tf/nlp/seg下
直接运行：

```python
python blistm_predicted.py

```
或者：
```python
#coding=utf-8
from __future__ import unicode_literals

from nlp.seg import blistm_predicted as seg

word = ['我', '们', '都', '是', '好', '孩', '子','，','，']
model=seg.load_model()
result=model.predict(word)
print (result)

```
